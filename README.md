Eshop MusicStore

La Aplicación MusicStore es una plataforma web completa diseñada para ofrecer a los usuarios una experiencia de compra fluida. Cuenta con una interfaz fácil de usar, una navegación robusta y una amplia variedad de productos para elegir. La aplicación está construida utilizando React, Material UI y Vite, lo que garantiza una arquitectura rápida, escalable y fácil de mantener.


Características

Autenticación y autorización de usuarios

Catálogo de productos con filtrado y ordenamiento

Carrito de compras con gestión de cantidades

Lista de favoritos con persistencia

Diseño responsive para dispositivos móviles y de escritorio

Enrutamiento del lado del cliente con React Router

Gestión de estado usando Context API


Tecnologías Utilizadas

Frontend: React, Material UI, React Router

Backend: Ninguno (solo del lado del cliente)

Herramienta de construcción: Vite

Base de datos: Ninguna (solo del lado del cliente)

Herramientas de IA: Ninguna

Framework de pruebas: Ninguno


Instalación

Para comenzar con el proyecto, sigue estos pasos:

Clona el repositorio usando git clone

Instala las dependencias usando npm install

Inicia el servidor de desarrollo usando npm run dev


Uso

Abre la aplicación en un navegador web

Navega al catálogo de productos y agrega productos al carrito de compras

Visualiza el carrito y gestiona las cantidades

Agrega productos a la lista de favoritos

Inicia y cierra sesión en la aplicación
